数字音效处理器

项目介绍

数字音效处理器是一个基于软件的音效处理工具，旨在实现各种语音信号的音效处理。该项目有以下要求：

  • 输入语音信号源为实际环境采集的语音。
  • 至少实现以下三种音效处理功能：
    1. 强噪声下的语音降噪。
    2. 语音与音乐、语音与语音的混合分离。
    3. 项目组由5人组成的语音识别。
  • 鼓励设计并实现其他音效处理功能。
  • 提供用户界面以进行人机交互，并支持扬声器/耳机输出音效。

项目演示

  • 音频录制
  • 音频播放
  • 语音降噪
  • 语音分离
  • 语音识别

音频播放

导入模块

  • time:用于处理时间
  • wave:用于处理.wav音频文件
  • tkinter:用于创建图形用户界面
  • pyaudio:用于音频播放

创建函数

  • stop_play:停止音频播放并关闭音频流
  • play_audio:在播放状态下，不断从音频文件中读取帧并通过音频流播放
  • toggle_play:处理播放按钮的点击事件，切换播放和暂停状态
  • update_time:更新界面上的已播放时间
  • pause_audio:暂停音频播放
  • audioPlayer_interface:创建用户界面，包括播放按钮、停止按钮和时间标签
